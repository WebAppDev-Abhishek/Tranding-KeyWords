Google Trends Clone (Node.js Backend & Frontend)

This project is a simplified clone of the Google Trends dashboard, demonstrating real-time trending keyword display and historical search interest visualization. It features a Node.js Express backend to serve static files and provide API endpoints for data, and a pure HTML, CSS (Tailwind CSS), and JavaScript frontend for the user interface.

Note: The trending and historical data are currently simulated on the backend. To get actual real-time data from the internet, you would need to integrate with external APIs (e.g., Google Trends API, Twitter API) and handle their authentication and rate limits.

Features

  • Trending Keywords List: Displays a simulated list of top trending keywords.
  • Search Functionality: Filter the trending keywords list, or if a keyword is not trending, display its simulated historical search interest.
  • Historical Trend Graph: Visualizes the simulated historical search interest for a given keyword using Chart.js.
  • "Back to Trending Keywords" Button: Easily switch back from the graph view to the trending list.
  • Responsive Design: Built with Tailwind CSS for a mobile-first approach.
  • Structured Project: Separated HTML, CSS, and JavaScript files for better organization.
  • Node.js Backend: Serves the frontend assets and provides API endpoints for data.

Project Structure

Setup and Running the Project

Follow these steps to get the project up and running on your local machine.

Prerequisites

Before you begin, ensure you have Node.js and npm (Node Package Manager) installed on your system.

  • Node.js & npm: Download and install from nodejs.org.

Installation

  1. Clone the Repository (or create the files manually):
    If you're getting this from a Git repository, clone it: 
    git clone <your-repository-url>
cd google-trends-clone
    If you've received the files directly, create a folder (e.g., google-trends-clone) and place index.html, style.css, script.js, and server.js inside it. Then navigate into that folder: 
    cd google-trends-clone
  2. Initialize Node.js Project:
    If you haven't already, initialize a Node.js project. This will create a package.json file. 
    npm init -y
  3. Install Dependencies:
    Install the necessary Node.js packages (express for the web server and cors for cross-origin requests). 
    npm install express cors

Running the Server

  1. Start the Node.js Server:
    In your terminal, from the google-trends-clone directory, run: 
    node server.js
    You should see output similar to this: 
    Server running on http://localhost:3000
Open your browser and navigate to this address.

Accessing the Application

  1. Open in Browser:
    Open your web browser and navigate to the address displayed in the terminal, typically: 
    http://localhost:3000

Usage

  • Trending Keywords: The homepage will display a list of simulated trending keywords.
  • Search: Use the search bar at the top to filter the trending keywords.
  • Historical Graph: If you search for a keyword that is not in the current trending list, the application will switch to a graph view, showing a simulated historical trend for that word.
  • Back Button: When viewing a historical graph, click the "Back to Trending Keywords" button to return to the main trending list.

Future Enhancements (Real-time Data)

To make this application truly reflect real-time internet trends, you would need to replace the simulated data logic in server.js with actual API calls to external services. Here are some ideas:

  • Google Trends API: For official Google search trend data. (Requires API key and adherence to Google's API terms).
  • Twitter API: To fetch trending hashtags or topics.
  • News API: To get trending news headlines and extract keywords.

Remember to handle API keys securely (e.g., using environment variables) and respect API rate limits.
